Class: Keyboard

Keyboard

new Keyboard(opts)

Creates a new instance of the Keyboard class.
Parameters:
Name Type Description
opts object the options to use to initialize the instance.
Properties:
Name Type Description
element object a reference to the root DOM element of the keyboard instance.
capsLock boolean a boolean flag which is true when caps lock is active, false otherwise.
shift boolean a boolean flat which is true when the shift key is active, false otherwise.
input object a reference to the input field within the keyboard instance.
layout object the current layout of the keyboard, this is used to provide configurable keyboard layouts.

render

Renders the keyboard on screen, the method generates all the element required by the keyboard object, and then invokes _attachEvents method to attach the event listeners to the keys and input field.

setLayout

A method to set the layout of the keyboard. This can be used to provide layouts for other languages. Note that this method removes the root element of the Keyboard instance and issues a render call to re-render the keyboard with the new layout.
